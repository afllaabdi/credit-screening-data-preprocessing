# Credit Screening Data Preprocessing

This project demonstrates a data preprocessing pipeline using the **Credit Screening dataset from the UCI Machine Learning Repository**.

## Workflow

1. Dataset loading using Pandas
2. Data inspection and type analysis
3. Handling missing values
4. Median imputation for numerical columns
5. Mode imputation for categorical columns
6. Data visualization using Seaborn (Boxplot)
7. One-hot encoding for categorical features
8. Feature scaling using MinMaxScaler

## Technologies

Python
Pandas
NumPy
Seaborn
Matplotlib
Scikit-learn

## Objective

To understand how raw datasets are cleaned, transformed, and prepared before applying machine learning models.
